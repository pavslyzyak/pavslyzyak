## Featured Project

### Northbridge Academy IT Service Desk Lab

A complete simulated IT support and networking environment built around a fictional UK secondary school and sixth form.

The project combines:

- GLPI service desk administration
- Docker deployment
- User, group, location and asset management
- Ticket categories, templates and solution types
- Priority matrix and P1-P4 service levels
- Incident and service-request workflows
- Requester approval and ticket closure
- Cisco Packet Tracer network design
- VLAN segmentation
- Router-on-a-stick
- DHCP relay
- ACLs
- Switch hardening
- Wireless testing
- Technical reporting and evidence collection

The project includes five completed support scenarios:

1. Password reset and account unlock
2. Microsoft Visio installation request
3. Staff Wi-Fi connectivity issue
4. Staff Room printer offline
5. Classroom projector displaying no image

Repository:
